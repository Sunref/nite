\section{Introdução}

\begin{frame}

    \frametitle{Introdução}

    O constante avanço tecnológico impulsiona o desenvolvimento de novas ferramentas e técnicas para a programação, levando desenvolvedores a buscar ferramentas que melhor lhe atendam. O Nano-Inspired Text Editor (NITE) surge nesse contexto como um editor de linha de comando simples, compacto e inspirado em soluções modernas, buscando combinar eficiência e praticidade. Este trabalho analisa editores e bibliotecas relacionadas ao NITE, destacando suas funcionalidades e contribuições para o desenvolvimento de ferramentas baseadas em terminal.

\end{frame}

\begin{frame}

    \frametitle{Objetivos}

    \begin{itemize}

	    \item Objetivo Geral
        \begin{itemize}
            \item Desenvolver um editor de texto executado em ``modo texto''/``linha de comando'', multiplataforma, baseado em Nano.
        \end{itemize}

        \item{Objetivos Específicos}
        \begin{itemize}
	       	\item Mapear e descrever ferramentas e bibliotecas;
	        \item Identificar boas práticas;
	        \item Melhorias do NITE a partir dessas ferramentas;
	        \item Desenvolver a aplicação final.
        \end{itemize}

    \end{itemize}

\end{frame}
